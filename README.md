# AI启示 - 轻量决策辅助工具

## 🌟 功能介绍

这是一个基于梅花易数原理的智能决策辅助工具，帮助您在困惑时获得内心启示。

### 核心功能

🎯 **智能场景分类**
- 职场选择：事业发展、工作决策
- 情感关系：感情问题、人际交往  
- 日常琐事：生活决策、选择困难

🔮 **梅花易数占卜**
- 基于时间、事件、随机数进行推算
- 齿轮式数字选择器，操作直观
- 详细的卦象解读和指导建议

📚 **历史记录**
- 自动保存每次咨询记录
- 支持时间排序（最新在前/最旧在前）
- 分类查看不同场景的决策历史

❤️ **收藏功能**
- 收藏重要的启示内容
- 快速查看收藏的智慧

## 🚀 使用方法

### 方法一：直接打开（推荐）
1. 双击 `index.html` 文件
2. 在浏览器中直接使用

### 方法二：本地服务器
1. 在项目文件夹中打开终端
2. 运行命令：`python3 -m http.server 8000`
3. 浏览器访问：`http://localhost:8000`

## 📱 支持平台

- ✅ 电脑浏览器（Chrome、Safari、Firefox、Edge）
- ✅ 手机浏览器（iOS Safari、Android Chrome）
- ✅ 平板设备

## 💾 数据存储

- 所有数据保存在本地浏览器
- 隐私安全，不会上传到服务器
- 清除浏览器数据会丢失历史记录

## 🎮 使用流程

1. **选择场景**：职场/情感/日常
2. **输入问题**：描述您的具体困惑
3. **获取启示**：AI生成个性化指导
4. **保存收藏**：保存重要的启示内容
5. **查看历史**：回顾过往的决策记录

## 🌸 梅花易数功能

1. **选择预测时间**：输入或点击"当前时间"
2. **调节随机数**：点击齿轮选择三位数（000-999）
3. **描述事件**：详细说明要预测的事情
4. **开始推断**：获得基于梅花易数的专业解读

## 🔧 技术说明

- 纯前端实现，无需网络连接
- 响应式设计，适配各种屏幕
- 现代Web技术，流畅的用户体验

## 📞 使用建议

- 保持问题具体明确，效果更佳
- 重要决策建议多参考几次
- 启示仅供参考，最终决定权在您手中
